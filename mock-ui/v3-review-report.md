## 一、设计规范合规性

### 通过项

- 字体：全部使用 Inter
- CSS 自定义属性（design tokens）：所有页面使用 `:root` 变量，为暗色模式预留
- 颜色：`--primary: #f97316`、`--accent: #92400e`、`--bg: #faf9f6`、`--surface: #fffffe`、`--text: #292524`、`--muted: #78716c` 全部正确
- 卡片圆角 16px、按钮圆角 12px、徽章圆角 8px 正确
- Anki 四档颜色：Again `#ef4444`、Hard `#f97316`、Good `#22c55e`、Easy `#3b82f6` 正确
- 卡片状态配色：新卡蓝、学习中红、待复习绿，色值正确
- 响应式导航：桌面顶部 + 手机底部，768px 断点，正确
- 底部导航有 SVG 图标 + 文字标签
- 导航栏 `backdrop-filter: blur(12px)` 玻璃拟态效果
- 按钮 hover/active 动效 150ms ease-out
- 首页 Bento grid 布局

## 四、总结

**整体评价：质量很高。** 5 个页面覆盖了需求文档所有核心功能模块（§3.1-3.5, §4, §11），设计规范基本遵循，交互逻辑完整且可实际操作。

### 核心亮点

- Session 页的 AI 评分 + 手动评级并行流程实现最到位
- 卡片组管理页 CRUD + 业务规则（默认组不可删、非空组不可删）完整
- 所有页面的空状态切换机制统一且方便评审
- CSS 自定义属性和响应式布局一致性好

### 进入 React 阶段前建议修复的 3 件事

1. review.html 的静态预览卡片加"仅为示意"标注，避免误导
2. 统一阴影和圆角的微小差异
3. home.html 空状态下"开始复习"按钮的处理

## 手动 Review 发现的问题（2026-02-27）

原始问题记录：
1. 记录Tab、复习Tab顶部菜单栏长度与卡片组不一致（卡片组更长）
2. 4个Tab页顶部菜单栏、内容区宽度需统一，按卡片组宽度为准
3. 切换首页↔记录Tab时整体内容右移约5-10px

**Playwright 实测根因（修复前）：**

| 页面 | topnav-inner | main | bodyClientW | 有滚动条 |
|------|-------------|------|-------------|---------|
| home | 960px | 960px | 1366px | No |
| record | 960px | 640px | 1351px | Yes |
| review | 960px | 960px | 1351px | Yes |
| decks | 1100px | 1100px | 1366px | No |
| session | 980px | 980px | 1366px | No |

- 问题1&2：decks 使用 1100px，其余用 960px / 640px / 980px
- 问题3：record/review 有垂直滚动条导致 bodyClientWidth 比 home/decks 少 15px，nav 居中位置偏移约 7-8px

**修复方案：**
- 所有页面 `.topnav-inner` 和 `.main` 统一改为 `max-width: 1100px`
- record 页 `.main` 内增加 `.content-narrow { max-width: 640px; margin: 0 auto; }` 内层容器收窄内容
- 所有页面 `<style>` 中加入 `html { scrollbar-gutter: stable; }` 稳定滚动条槽位

**Playwright 实测验证（修复后）：**

| 页面 | topnav-inner | main | content-narrow | bodyClientW |
|------|-------------|------|---------------|-------------|
| home | 1100px ✅ | 1100px ✅ | — | 1351px ✅ |
| record | 1100px ✅ | 1100px ✅ | 640px ✅ | 1351px ✅ |
| review | 1100px ✅ | 1100px ✅ | — | 1351px ✅ |
| decks | 1100px ✅ | 1100px ✅ | — | 1351px ✅ |
| session | 1100px ✅ | 1100px ✅ | — | 1351px ✅ |

全部 bodyClientWidth 一致，切 Tab 不再横向跳动。三个问题均已修复。
